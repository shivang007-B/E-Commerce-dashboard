## 🛠️ TECH STACK & ARCHITECTURE

* **Framework**: Next.js 15 with App Router for optimized routing and server components.
* **Styling**: Tailwind CSS utilizing custom neon-glow utilities and glassmorphism.
* **Database**: MongoDB via Mongoose, using a consolidated connection utility to prevent production bottlenecks.
* **Authentication**: NextAuth.js supporting Google OAuth and secure manual credentials.
* **Payments**: Integrated Razorpay gateway with SHA256 HMAC signature verification.
* **State Management**: Zustand-powered "Tactical Cart" for real-time inventory tracking.
